# Event Planner

The project was created using [Vite](https://vitejs.dev/).

## About

The project was created in order to perform a test task.

### Technologies used in the project:

- React.js
- Firebase
- PropTypes
- Redux, Redux-persist
- React-datepicker
- Keen-slider (timepicker)
- React-icons
- Styled-components
